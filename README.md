## 📌 Overview
This project is a **Playwright Automation Framework** built using the **Page Object Model (POM)** design pattern.

It supports **multi-application testing (Admin + Customer)** and is designed for scalability and maintainability.

---

## ✨ Features
✅ Page Object Model (POM)  
✅ Multi-app testing (Admin & Customer)  
✅ Config-driven URL handling  
✅ Reusable utilities  
✅ Parallel execution support  
✅ HTML test reports
